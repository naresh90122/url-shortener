This project is used to short the url like bitly. It is use full when we share it in encrypted way but short. This Project was made of Node JS, EJS, JavaScript, HTML, CSS and MongoDB Database. In this Project we have sigin page and login page. We have implemented in this project middleware, auth, session and cookie features. In this project, you can create short url which is convert your long url into short url. We can track how many time this short url hit by user.
